# 题目

从`有序`顺序表中删除其值在给定值s与t之间(要求s<\t)的所有元素，如果s或t不合理或顺序表为空，则显示出错误信息并退出运行。

# 算法思路

因为题目中说的是有序表，所以删除的元素必然是`相连的整体`。

# 算法

```c
bool Del_st1(Sqlist &L,Elemtype s,Elemtype t){
	int k=0;
	if(s>t||L.length<0)
		return false;
	for(int i=0;i<L.length;i++){
		if(L.data[i]>=s && L.data[i]<=t)		
			k++;
		else
			L.data[i-k]=L.data[i];
	}
	L.length=L.length-k;
	return true;
}
```
